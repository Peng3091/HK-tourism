## 项目介绍
整合Vue.js,  Mapbox, Koa.js, MySQL和MongoDB等多种框架和技术, 通过前端, 服务端和数据端的有效结合, 设计并实现了包含景点查询、行程规划、娱乐推荐和咨询评价等功能的香港智慧旅游系统. 与以往的旅游信息系统相比, 该系统数据管理更灵活、扩展性更好、交互性更强、行程规划功能更加完善.
